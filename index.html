Papiro

Papiro é uma aplicação web desenvolvida para gerenciar o controle de pagantes mensais, focada em facilitar a gestão financeira de assinaturas recorrentes. A plataforma permite que os administradores visualizem rapidamente o status de pagamento dos clientes, organizando-os de forma eficiente e intuitiva.

Stacks usadas: Node.js, Firebase, Typescript, Nexts.js, Sass
